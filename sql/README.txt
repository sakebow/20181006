-- GENERAL
In this folder, there're some sql scripts. But the lastest one is version 4
-- 总体介绍
在这个文件夹下有很多sql脚本。最新的是第4版。

-- WHAT'S IN EACH FILE?
All tables, primary keys, foreign keys, along with check and default are defined in "aspDBV4".
While there's something more might be in need in "addonsDB", though there's nothing useful remaining after updating several times already.
-- 这些文件夹里都有些什么？
所有的表格、主键、外键，还有约束、默认值都在"aspDBV4.sql"中。
一些可能会用上的东西存在"addonsDB.sql"中，但经过多次版本更新就已经没什么有用的了。

-- HOW TO USE IT?
If you haven't build up your database, just run "finaltest.sql" in your SQL Server 2014 Enterprise, which can be downloaded in MSDN.
Although many details haven't completed yet, they are still enough for a student management system.
-- 怎么使用？
如果你还没有建好自己的数据库，就只需要在SQLServer2014企业版下跑一遍"aspDBV4.sql"就好了，可以在MSDN中下载该数据库软件。
虽然还是有很多细节没有完善，但是现有的依然能够满足学生管理系统的需求。

-- SOMETHING WORTH METHONING
All tables and fields are defined in Chinese. 
You can change them into English or any language else as you like
Some of all the fields might be useless, you can delete them to fit your requirement
-- 一些需要说明的
所有的表、字段全都是用中文定义的。
你可以将他们改成英文或者其他你喜欢的语言。
其中一些字段也许是没有用的，你可以删掉并按照自己的需求更改。
